ðŸ“¡ Overview#

This site defines custom metadata to improve how content appears in:

  • Social media link previews (Twitter, Discord, LinkedIn)
  • Search engine snippets (SEO basics)
  • Browser tabs and mobile sharing

Metadata is injected via Hugo partials and frontmatter using Open Graph, Twitter Card, and standard HTML meta tags.

ðŸ–¼ Dynamic Social Preview Image#

You can define a global image preview via config.toml:

[params]
  images = ["/banner.png"]

This is referenced in extend_head.html:

{{ with .Site.Params.images }}
  <meta property="og:image" content="{{ index . 0 | relURL }}" />
{{ end }}

You can later allow per-page override by adding a images array in frontmatter:

images: ["/projects/secure-home-lab/preview.png"]

ðŸ§© Additional Metadata from PaperMod#

The theme injects these from core files:

  • <meta name="description">

  • <meta name="author">

  • Favicon links:

    • /favicon.png
    • /favicon-16x16.png
    • /apple-touch-icon.png

These are all defined in config.toml:

[params]
  author = "Alex Baker"

[params.assets]
  favicon = "/favicon.png"
