ðŸ§© 1. Homepage Override: layouts/index.html#

Used to inject a theme-responsive banner image above the homepage content.

{{ define "main" }}
<main>
  <p>
    <img id="hero-banner"
         src="/banner.png"
         data-dark="/banner.png"
         data-light="/banner-inverted.png"
         alt="alexbakertech banner"
         style="width:100%; border-radius:8px;" />
  </p>
  {{ .Content }}
</main>
{{ end }}

This file replaces the default list.html rendering for the homepage, allowing custom HTML and image logic to be embedded directly.

ðŸ–¼ 2. Footer Script Override: layouts/partials/extend_footer.html#

Used to sync the banner image with the PaperMod theme toggle:

<script>
  function updateBannerByTheme() {
    const banner = document.getElementById("hero-banner");
    if (!banner) return;
    const theme = localStorage.getItem("pref-theme") || 
      (document.body.classList.contains("dark") ? "dark" : "light");
    banner.src = (theme === "dark") ? banner.dataset.dark : banner.dataset.light;
  }

  document.getElementById("theme-toggle")?.addEventListener("click", () => {
    setTimeout(updateBannerByTheme, 10);
  });

  updateBannerByTheme();
</script>

This script runs after page load and updates the image path based on theme toggle state, not system preference.
